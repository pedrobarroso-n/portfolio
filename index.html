Nós de Arte

Este website é baseado em um projeto anterior, desenvolvido na wix.com como projeto de exibição durante um evento da faculdade. Um breve resumo para contexto, a Nós de Arte é uma pequeno conjunto de artesãs de minha cidade, este projeto visava criar um meio onde as mesmas poderiam apresentar seus trabalhos e vender seus produtos.

Controle de Vendas

Este projeto consiste em um sistema simples de controle de vendas usando a linguagem Python fortalecido usando a base de dados Sqlite3 junto do microframework Flask, a interface permite realizar funções simples como: adicionar, editar e excluir demais registros de 3 tabelas relacionadas.
